# Assistance

Si vous avez un soucis pour accéder au calculateur, il est possible que suite à plusieurs échec de mot de passe votre adresse IP ait été bannie.

Avant de créer un ticket, merci de bien consulter la page de maintenance : <https://romeo.univ-reims.fr/maintenance>

Si vous avez un problème sur le calculateur, une question, ou une remarque concernant la sécurité informatique, n'hésitez pas à soumettre un ticket dans le système prévu à cet effet pour nous contacter : [Support ROMEO](https://romeo-ticket.univ-reims.fr/)
