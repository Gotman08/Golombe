# Accèder au portail ROMEO

## Etape 1 : Accès au portail ROMEO[â](#etape-1--accès-au-portail-romeo "Lien direct vers Etape 1 : Accès au portail ROMEO")

Le portail ROMEO est le point d'entrée pour accéder à l'ensemble des services ROMEO.

L'accès au portail vous permet de gérer vos projets ainsi que vos clés SSH d'accès aux ressources matérielles.

> ⚠️ **Warning**
>
> attention

En fonction de votre établissement d'origine et de votre statut, la procédure d'accès au portal est différente : personnel ou étudiant de l'URCA (accès CAS URCA) ou extérieur, académique ou entreprise (accès via magic link)

Seuls les personnels permanents de l'URCA et des établissement partenaires (UL, UNISTRA, UTT, CHU Reims) peuvent créer un projet. Les autres utilisateurs peuvent participer à un projet.


- Autres universités (via EDUGAIN)
- CAS URCA
- Autres, via MAGIC-Link

### Accès pour les extérieurs à l'URCA[â](#accès-pour-les-extérieurs-à-lurca "Lien direct vers Accès pour les extérieurs à l'URCA")

#### Identification sur le portail via EDUGAIN[â](#identification-sur-le-portail-via-edugain "Lien direct vers Identification sur le portail via EDUGAIN")

EduGAIN est un service d'interconnexion des fédérations éducation/recherche européen.

1. Bien lire la [Charte d'utilisation](2.5.charte.md) !
2. Aller sur le portail de Romeo sur [<https://romeo.univ-reims.fr/portal>].
3. Se connecter avec vos identifiants habituels via le bouton "EDUGAIN", qui vous renvoie vers l'outil de fédération de l'URCA [<https://iam.univ-reims.fr>] puis via le CAS de votre établissement.
4. Sur l'outil de fédération de l'URCA [<https://iam.univ-reims.fr>], veuillez à bien expliquer qui vous êtes (laboratoire, lien vers le site du laboratoire ou votre nom apparrait, responsable pour les non-permanents, projet, ...)
5. Votre compte sur l'outil de fédération de l'URCA [<https://iam.univ-reims.fr>] necessite une validation manuelle et peut necessiter des informations complémentaires, surveillez vos emails (et spams).
6. Une fois votre compte validé sur l'outil de fédération de l'URCA [<https://iam.univ-reims.fr>] vous pouvez vous connecter sur le portail ROMEO [<https://romeo.univ-reims.fr/portal>].

### Accès CAS URCA pour les personnels et étudiants de l'URCA[â](#accès-cas-urca-pour-les-personnels-et-étudiants-de-lurca "Lien direct vers Accès CAS URCA pour les personnels et étudiants de l'URCA")

Votre connexion au portail ROMEO utilise maintenant le CAS de l'URCA.

> ⚠️ **Warning**
>
> attention

Si vous aviez un compte sur l'ancien calculateur Romeo 2018, celui-ci n'est plus d'actualité. Un nouveau compte est créé sur la machine une fois un projet rejoint et une clé associée. Suivez la documentation ci-dessous même si vous aviez un compte sur le précédent supercalculateur.

#### Identification sur le portail[â](#identification-sur-le-portail "Lien direct vers Identification sur le portail")

1. Bien lire la [Charte d'utilisation](2.5.charte.md) !
2. Aller sur le portail de Romeo sur [<https://romeo.univ-reims.fr/portal>].
3. Se connecter avec vos identifiants URCA via le bouton "Authentification URCA", puis via le CAS de l'URCA.
4. C'est bon, vous êtes connectés au portail ROMEO !

### Accès en dehors du périmètre défini.[â](#accès-en-dehors-du-périmètre-défini "Lien direct vers Accès en dehors du périmètre défini.")

Vous souhaitez accéder aux ressources ROMEO mais n'êtes pas chercheur d'un établissement membre d'EDUGAIN ? C'est possible sur invitation ou sur critères scientifique ou pour les entreprises gràce à notre programme d'accompagnement.

1. Bien lire la [Charte d'utilisation](2.5.charte.md) !
2. Faire une demande d'accès au portail ROMEO via un ticket (<https://romeo-ticket.univ-reims.fr/>) dans la catégorie "Accéder au portail ROMEO pour les extérieurs à l'URCA". Veuillez à bien remplir l'ensemble des champs et à vérifier vos emails (peut-être vos spams aussi)
3. Attendre le traitement du ticket. Vous recevrez un email dès que le ticket sera traité.
4. Aller sur le portail de ROMEO sur [<https://romeo.univ-reims.fr/portal>].
5. Se connecter via le bouton "MagicLink" puis renseigner votre email professionnel utilisé dans le ticket
6. Vous recevrez un email contenant un lien unique valide pendant 10 minutes
7. Si un message d'attente de validation apparait, c'est que le ticket n'a pas été traité ou que l'adresse email utilisée n'est pas la bonne.
8. C'est bon, vous êtes connecté au portail ROMEO!

### Première connexion sur le portail ROMEO[â](#première-connexion-sur-le-portail-romeo "Lien direct vers Première connexion sur le portail ROMEO")

##### Définir un login préféré[â](#définir-un-login-préféré "Lien direct vers Définir un login préféré")

Vous pouvez définir un nom de compte préféré utilisé pour vous connecter aux ressources de Romeo dans votre profil. Vous pouvez y accéder en cliquant sur votre adresse mail en haut à droite de l'écran, puis sur « Mon profil ». Pour définir votre login préféré, renseignez-le dans l'entrée dédiée puis cliquez sur « Apply ».

Ce login sera utilisé pour créer votre compte sur les machines quand cela est possible, c'est avec celui-ci que vous pourrez vous connecter.

## Étape 2 : Créer ou Rejoindre un projet[â](#étape-2--créer-ou-rejoindre-un-projet "Lien direct vers Étape 2 : Créer ou Rejoindre un projet")

> ⚠️ **Warning**
>
> attention

Seuls les utilisateurs avec des adresses email de personnel des universités du Grand-Est peuvent créer des projet.
Les autres utilisateurs peuvent être associés à des projets existants pour accéder aux services ROMEO

### Rejoindre un projet[â](#rejoindre-un-projet "Lien direct vers Rejoindre un projet")

Pour lancer des calculs sur les machines proposées par Romeo, vous aurez besoin que votre compte soit lié à un projet. Pour cela, il faut qu'un responsable de projet vous ajoute en tant que membre de projet, ou que vous demandiez la création d'un projet via le portail.

### Créer un projet[â](#créer-un-projet "Lien direct vers Créer un projet")

- Pour pouvoir créer un projet vous devez être personnel permanent de l'URCA ou d'une université partenaire (UTT, UNISTRA, UTT, CHU, UHA, CEA, ...)
- Dans le cas contraire, vous pouvez rejoindre un projet existant en y étant invité.
- Dans le menu principal du portail, cliquer sur « Tableau de bord », « Mes projets », puis « Nouveau projet ».
- Remplissez les informations nécessaires et sélectionnez la machine « Romeo2025 ».
- Vous devez ensuite **soumettre ce projet à l'équipe ROMEO** pour expertise scientifique et technique en cliquant sur le bouton associé, dans la page du projet.
- Allez ensuite dans la page du projet pour vérifier que vous y êtes présent en tant que membre, si ce n'est pas le cas vous devez vous y ajouter (à l'aide de la même adresse émail que celle que vous utilisez sur ce portail)
- Une fois le projet validé par l'équipe ROMEO, vous pourrez y ajouter des membres via leur adresse email.

## Étape 3 : Ajouter une clé de connexion SSH[â](#étape-3--ajouter-une-clé-de-connexion-ssh "Lien direct vers Étape 3 : Ajouter une clé de connexion SSH")

- La page d'accueil du portail (Tableau de bord), présente les projets pour lesquels vous être membre. (Il s'agit soit de projets dont vous être le propriétaire, soit de projets auxquels vous avez été invité)
- Pour chaque projet, est présenté son code (utilisé comme account pour soumettre des jobs ou pour les répertoires projets)
- Pour chaque projet, vous pouvez voir la machine associée ainsi que les clés et logins associés.

**IMPORTANT :** Veillez à réaliser cette étape *après* avoir rejoint un projet sur lequel déployer la ou les clés.

Voir [Se connecter aux ressources et gestion des clés ssh](ressources/connexion_ssh.md).

Une fois la clé ssh associée à une machine et un projet (validé), votre compte et vos accès seront créés sur la machine en question dans l'heure qui suit. Vous pourrez vous connecter via le nom de compte indiqué dans la page principale « Tableau de bord » du portail.
